# 版本历史

本文档记录了 EasyTier 节点管理系统的版本历史和更新内容。

---

## 版本命名规则

本项目采用语义化版本号（Semantic Versioning）：`主版本号.次版本号.修订号`

- **主版本号（Major）**：重大功能更新或不兼容的 API 修改
- **次版本号（Minor）**：向下兼容的功能性新增
- **修订号（Patch）**：向下兼容的问题修正

### v1.0.0（首次发布）

**发布日期**：2025-01-03

**主要内容**：
- 系统初始化功能
- 三级权限系统（超级管理员/普通管理员/普通用户）
- 系统设置管理（Resend 配置、网站配置）
- 用户管理功能
- 节点管理（CRUD）
- 节点上报 API
- 智能负载均衡查询
- 响应式前端界面

**核心功能**：

1. **用户系统**
   - 用户注册和登录
   - 邮箱验证（Resend）
   - 三级权限管理

2. **节点管理**
   - 节点信息管理
   - 节点上报 API
   - 智能负载均衡

3. **管理功能**
   - 管理员面板
   - 系统设置
   - 用户管理

4. **前端界面**
   - 响应式设计
   - 现代化 UI
   - 完整的用户交互
